# Walter & Walter Kft. Landing Page

Modern, reszponzív landing page a Walter & Walter Kft. számára.

## Funkciók

- ✅ Reszponzív design (mobil, tablet, desktop)
- ✅ Modern, professzionális megjelenés
- ✅ Hero szekció a cég bemutatásával
- ✅ Szolgáltatások szekció
- ✅ Rólunk szekció
- ✅ Kapcsolat űrlap
- ✅ Elérhetőségek megjelenítése
- ✅ Smooth scrolling navigáció
- ✅ Mobil menü

## Fájlok

- `index.html` - Fő HTML struktúra
- `styles.css` - Stílusok és reszponzív design
- `script.js` - JavaScript funkcionalitás (form kezelés, navigáció)

## Használat

Egyszerűen nyissa meg az `index.html` fájlt egy böngészőben.

## Form integráció

A jelenlegi implementációban a form adatok csak a böngészőben vannak kezelve. Éles környezetben szükséges a backend integráció:

1. Módosítsa a `script.js` fájlban a form submission részt
2. Adja hozzá a saját API endpoint-ot
3. Konfigurálja a szerver oldali email küldést

## Testreszabás

- Színek: Módosíthatók a CSS `:root` változóiban
- Tartalom: Módosítható az `index.html` fájlban
- Stílusok: Módosíthatók a `styles.css` fájlban
